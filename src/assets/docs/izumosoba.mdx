日本三大そば(わんこそば・戸隠そば・出雲そば)の一つに数えられる出雲の郷土料理です。

3段重ねになっているのが特徴です。

出雲大社周辺にいっぱい店があって父親いわく味に優劣があるそうですが、私には全くわかりません。

地元のスーパーならプラスチックのパックに入ったのがよく売っています。

出典：農林水産省「うちの郷土料理」
画像提供元 : 日本の食文化発信サイト「SHUN GATE」
